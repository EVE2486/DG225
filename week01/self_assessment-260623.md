> 958225 Digital Game Production — แบบฟอร์มบันทึกความคืบหน้าและการประเมินตนเองรายสัปดาห์

## ① สรุปเนื้อหาที่เรียน — ฉันได้เรียนรู้อะไรในสัปดาห์นี้?

= ได้เรียนรู้เรื่องความปลอดภัยของการสร้างบัญชีส่วนตัวบนคอมพิวเตอร์ และการลงโปรแกรม extentions ต่าง ๆ บน vs code

## ② สรุปการฝึกปฏิบัติ — ฉันได้ทดลองและฝึกเขียนโปรแกรมอะไรบ้าง?

= ได้ทดลองการสร้างบัญชีส่วนตัวของตัวเอง และฝึกเขียนโปรแกรมบน vs code

## ③ ปัญหา / ข้อสงสัย — สิ่งที่ยังไม่เข้าใจหรืออยากสำรวจต่อ

= เนื้อหาค่อนข้างมีรายละเอียดเยอะ ต้องใช้เวลาในการทำความเข้าใจ
